# Personal Project (개인 프로젝트)
-----


# 개요
* 프로젝트 : CRUD 를 이용한 간단한 게시판

* 일정 : 2023. 05. 16 ~ 2023. 05. 24

* 목적 : 학습 중인 스프링 프레임워크를 복습하고자 진행중
           
### 사용 기술 및 개발환경
   - Server : Tomcat9.0
   - DB : MySQL
   - Framework/flatform : Spring, jQuery, MyBatis
   - Programming Language : Java, HTML/CSS, JavaScript
   - Tool : IDE(IntelliJ), Github
   
### 구현 기능
   - ~~회원가입 / 로그인 (+ 아이디 저장 기능)~~
   - ~~페이징~~
   - ~~게시글 작성/수정/조회/삭제~~
   - ~~조회수~~

### 추후에 추가할 기능
   - 게시글 검색
   - 댓글
   - 마이 페이지 (본인이 작성한 게시글 조회 등)
   - ID/PW 찾기


### 추후에 학습이 더 필요한 부분
   - 스프링 및 Tiles 초기 설정
   - 파일 업로드 / 댓글 / 페이징
